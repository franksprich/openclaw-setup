##How to Use This SOP
3 steps. 5 minutes. Your AI assistant — fully configured.



###Step 1 — Get the File
Open the link below and click the "Raw" button at the top of the page. Then Select All (Ctrl+A / Cmd+A) and Copy (Ctrl+C / Cmd+C).

[OpenClaw_Master_SOP](OpenClaw_Master_SOP.md)


###Step 2 — Upload to OpenClaw
In OpenClaw, go to your workspace and paste the full text into a new message. OpenClaw will read the entire document, create all your system files (SOUL.md, AGENTS.md, MEMORY.md, HEARTBEAT.md, TOOLS.md, and 4 skills), then ask you 10 personalization questions.

**Alternative**: Save the raw text as a .md file and upload it directly as a document.

###Step 3 — Answer the Questions
OpenClaw will ask you 10 questions about your name, business, priorities, working style, and preferences. Answer them all in one message. It will then populate every placeholder, finalize your configuration, and run a system audit. That's it — you're done.

What Gets Configured
SOUL.md — Personality, response format, voice, security rules
AGENTS.md — Your identity, priorities, CEO frame, multi-model routing
MEMORY.md — Persistent memory system that compounds over time
HEARTBEAT.md — 15-min heartbeats + 4 daily/weekly cron jobs
4 Skills — Meeting Prep, System Audit, Security Audit, Context Loader
